# PAE Bird

PAE Bird is a macOS polynomial algebra engine designed for high-speed symbolic math and batch processing.

It expands, factors, solves, and processes large sets of expressions instantly—including worksheet generation for students and educators.

---

## 🚀 What it does

PAE Bird is built for fast algebra workflows:

- Expand polynomial expressions
- Factor and simplify equations
- Solve symbolic expressions
- Generate math worksheets
- Process 100+ expressions in bulk
- Run completely offline on Mac

---

## ⚡ Batch Processing Engine

Unlike traditional calculators, PAE Bird is designed for scale.

You can:
- Input single expressions
- Or process 100+ equations at once
- Generate structured outputs for study or teaching
