PAE Bird

A high-speed polynomial algebra engine for macOS. Expand, factor, solve, differentiate, and generate worksheets — instantly, offline, in bulk.

What PAE Bird does

📐

Polynomial Expansion

Expand any polynomial expression, including multi-variable and nested parentheses.

🔢

Factoring

Full factoring — difference of squares, trinomials, cubes, grouping, and more.

🔍

Solving

Solve symbolic expressions and equations step by step.

∂

Calculus

Differentiate and integrate polynomial expressions using standard rules.

📋

Batch Processing

Input 100+ expressions at once. Process them all in one click.

📄

Worksheet Generator

Generate numbered student and teacher worksheets ready to print or export.

See it in action

Built for real math workflows

One input box. Every algebraic operation. Batch process an entire problem set in seconds.

Expand Polynomials

Paste any number of expressions and expand them all at once. Results appear instantly in the Solutions pane.

Factor Expressions

Switch to factor mode with a single @:factor directive. Handles difference of squares, trinomials, cubes, and grouping.

Every Operation in One Batch

Mix expand, factor, solve, differentiate, and integrate in a single input using @: mode switches. PAE Bird handles each section correctly.

Worksheet Output

Generate formatted worksheets with numbered problems and solutions — ready for students or educators to print and use.

Built-in Command Reference

Every supported operation and @: directive documented inside the app. No external docs needed.

Microsoft Excel®

Coming to Excel

PAE Bird as Excel Functions

Use PAE Bird's math engine directly inside Excel cells — no app switching, no copy-paste. Solve, expand, factor, differentiate, and integrate as native spreadsheet formulas.

=PAE.SOLVE("x^2 - 5x + 6 = 0") =PAE.EXPAND("(2x-1)^3") =PAE.FACTOR("2x^2 + 6x + 4") =PAE.DIFFERENTIATE("x^3 + 2x","x") =PAE.INTEGRATE("3x^2","x")

All five PAE functions returning results live in Excel cells.

Native Excel formula syntax — type it just like any other function.

Generate Student and Teacher worksheets directly from the task pane.

Google Sheets™

Available as a Google Sheets™ Add-on

PAE Bird in Google Sheets™

Use PAE Bird's algebra engine as native Google Sheets™ formulas — solve equations, expand, factor, differentiate, and integrate right inside your spreadsheet. Powered by PAE Bird's cloud API; math expressions are processed securely and never stored.

Note: unlike the offline Mac app, the Google Sheets™ add-on sends formula inputs to PAE Bird's API to compute results.

=PAE_SOLVE("x^2 - 5x + 6 = 0") =PAE_EXPAND("(2x-1)^3") =PAE_FACTOR("2x^2 + 6x + 4") =PAE_DIFFERENTIATE("x^3 + 2x","x") =PAE_INTEGRATE("3x^2","x")

Results returned directly into Sheets cells. Formula bar shows the PAE Bird function.

All five operations available: solve, expand, factor, differentiate, integrate.

Native Sheets autocomplete surfaces all PAE Bird functions as you type.

About Stewart Breier

Work on PAE Bird began with hand coding, with the first few rounds programmed in LISP, before migrating to C++. Later developments increasingly incorporated AI. The first spark of a dream of a PAE Bird originated during my first youthful programming lessons with BASIC, where I was amazed at how I could automate Algebra.

When I am not programming PAE Bird, I am in the classroom as a Substitute Teacher, furthering my education at National University in their MSW program, submitting my poetry to publications, playing TTRPGs, painting digital art, or working on my Tai Chi.
