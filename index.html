Built for real math workflows

One input box. Every algebraic operation. Batch process an entire problem set in seconds.

Expand Polynomials

Paste any number of expressions and expand them all at once. Results appear instantly in the Solutions pane.

Factor Expressions

Switch to factor mode with a single @:factor directive. Handles difference of squares, trinomials, cubes, and grouping.

Every Operation in One Batch

Mix expand, factor, solve, differentiate, and integrate in a single input using @: mode switches. PAE Bird handles each section correctly.

Worksheet Output

Generate formatted worksheets with numbered problems and solutions — ready for students or educators to print and use.

Built-in Command Reference

Every supported operation and @: directive documented inside the app. No external docs needed.
